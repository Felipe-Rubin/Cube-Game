# Cube-Game
An OpenGL Cube battle game made for CG course 2018/1 PUCRS

## Features
- Multi-Player Game
- Player one uses mouse and keyboard (WASD)
- Player two uses a Joystick (can also use Arrow keys for testing)
- - May require drivers
